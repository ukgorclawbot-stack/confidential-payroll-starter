# Project Summary

## Project

Confidential Payroll Starter Pack for Zama

## One-paragraph summary

This repository is a documentation-first builder starter that explores how confidential payroll can be designed on top of Zama. It focuses on one practical question: how payroll coordination can happen onchain without exposing sensitive compensation data publicly. The current version includes a concept note, architecture overview, builder tutorial, and sample payroll flow, with room for a future prototype.

## Problem

Traditional onchain systems are transparent by default. That is useful for many applications, but payroll has different requirements. A transparent payroll system can reveal salary levels, bonus structure, internal hierarchy, contractor relationships, and treasury behavior tied to operations.

## Why this matters

Payroll is one of the clearest real-world examples of why confidential blockchain infrastructure matters. It requires both onchain coordination and meaningful privacy. That makes it a strong entry point for builders exploring confidential applications.

## Why Zama

Zama makes it possible to think beyond the usual tradeoff between offchain privacy and onchain execution. It creates a path for applications where execution and workflow state can remain onchain while sensitive values stay protected.
